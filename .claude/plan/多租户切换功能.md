# 多租户切换功能 - 实施计划

**方案**：tRPC 组织路由 + TeamSwitcher 组件集成
**评估日期**：2026-01-15
**预期影响**：高（核心多租户体验改进）

---

## 📋 需求概述

在 Admin UI 左上角添加多租户切换功能，使用 shadcn sidebar-07 的 TeamSwitcher 组件模板。

**用户交互**：
- 展开时：显示组织 Logo + 名称
- 收起时：仅显示 Logo
- 切换后：导航到首页 `/` 并清理所有缓存

**架构约束**：
- 多租户隔离（防止跨租户数据泄露）
- 权限绑定（所有权限基于租户）
- 成员验证（禁止访问非成员组织）
- 审计日志（记录切换操作）

### API 设计

#### 1. `listMine` - 获取用户所属组织列表

**输入**：无
**权限**：`protectedProcedure`（需登录）

**查询逻辑**：
```sql
SELECT
  organization.id, name, slug, logo, createdAt,
  member.role
FROM organization
INNER JOIN member ON member.organizationId = organization.id
WHERE member.userId = :userId
  AND member.status != 'banned'
ORDER BY organization.name ASC
```

**返回类型**：
```typescript
{
  organizations: Array<{
    id: string;
    name: string;
    slug: string;
    logo: string | null;
    role: string; // 'OWNER' | 'ADMIN' | 'MEMBER'
    createdAt: Date;
  }>
}
```

**错误处理**：
- 未登录 → `TRPCError(UNAUTHORIZED, "Authentication required")`
- 查询失败 → `TRPCError(INTERNAL_SERVER_ERROR, "Failed to load organizations")`

---

#### 2. `setActive` - 切换活跃组织

**输入**：
```typescript
z.object({
  organizationId: z.string().uuid()
})
```

**验证流程**（严格执行）：
1. ✅ 验证 `ctx.userId` 存在
2. ✅ 查询 `member` 表，确认用户是该组织成员
3. ✅ 检查 `member.status != 'banned'` 且 `banExpires < now`
4. ✅ 调用 Better Auth API：`auth.api.setActiveOrganization({ organizationId })`
5. ✅ 刷新 session 确认 `activeOrganizationId` 更新
6. ✅ 记录审计日志：
   ```typescript
   {
     action: 'organization.set_active',
     actorId: userId,
     tenantId: organizationId,
     metadata: { organizationId }
   }
   ```
7. ✅ 清理权限缓存（可选，视现有缓存机制而定）

**返回类型**：
```typescript
{
  activeOrganizationId: string;
  organization: {
    id: string;
    name: string;
    slug: string;
    logo: string | null;
    role: string;
  }
}
```

**错误处理**：
- 未登录 → `TRPCError(UNAUTHORIZED, "Authentication required")`
- 非成员 → `TRPCError(FORBIDDEN, "Not a member of this organization")`
- 被封禁 → `TRPCError(FORBIDDEN, "Organization access is banned")`
- Better Auth 调用失败 → `TRPCError(BAD_GATEWAY, "Failed to set active organization")`
- 其他 → `TRPCError(INTERNAL_SERVER_ERROR, "Unable to switch organization")`

---

### 安全检查清单

| 检查项 | 验证内容 |
|--------|----------|
| **身份验证** | 两个 procedure 都使用 `protectedProcedure` |
| **成员验证** | `setActive` 必须验证 `member` 关系存在 |
| **封禁检查** | 过滤 `status='banned'` 或 `banExpires > now` |
| **跨租户防护** | 仅允许切换到已加入的组织 |
| **审计日志** | 记录切换操作的 `actorId` 和 `tenantId` |
| **缓存清理** | 切换后触发权限/菜单缓存失效 |

#### 1. `TeamSwitcher` - 纯展示组件

**重构为受控组件**（移除内部 state）：

```typescript
export interface TeamSwitcherProps {
  teams: Array<{
    id: string;
    name: string;
    logo: string; // URL 或回退字符
    plan: string; // 'Free' | 'Pro' | 'Enterprise'
  }>;
  activeTeamId?: string;
  isLoading: boolean;
  onSwitchTeam: (teamId: string) => Promise<void>;
  onCreateTeam: () => void;
}

export function TeamSwitcher({
  teams,
  activeTeamId,
  isLoading,
  onSwitchTeam,
  onCreateTeam
}: TeamSwitcherProps) {
  const { isMobile, state } = useSidebar();
  const isCollapsed = state === 'collapsed';
  const activeTeam = teams.find(t => t.id === activeTeamId);

  // 加载态
  if (isLoading) {
    return (
      <SidebarMenu>
        <SidebarMenuItem>
          <SidebarMenuButton size="lg">
            <Skeleton className="size-8 rounded-lg" />
            {!isCollapsed && (
              <>
                <Skeleton className="h-4 w-24" />
                <Skeleton className="h-3 w-16" />
              </>
            )}
          </SidebarMenuButton>
        </SidebarMenuItem>
      </SidebarMenu>
    );
  }

  // 空状态（无组织）
  if (teams.length === 0) {
    return (
      <SidebarMenu>
        <SidebarMenuItem>
          <SidebarMenuButton size="lg" onClick={onCreateTeam}>
            <Plus className="size-4" />
            {!isCollapsed && <span>Create Organization</span>}
          </SidebarMenuButton>
        </SidebarMenuItem>
      </SidebarMenu>
    );
  }

  return (
    <SidebarMenu>
      <SidebarMenuItem>
        <DropdownMenu>
          <DropdownMenuTrigger asChild>
            <SidebarMenuButton
              size="lg"
              className="data-[state=open]:bg-sidebar-accent"
            >
              {/* Logo */}
              <div className="flex aspect-square size-8 items-center justify-center rounded-lg bg-sidebar-primary text-sidebar-primary-foreground">
                {activeTeam?.logo ? (
                  <img src={activeTeam.logo} alt={activeTeam.name} className="size-4" />
                ) : (
                  <span>{activeTeam?.name[0]}</span>
                )}
              </div>
              {/* 展开时显示名称和套餐 */}
              {!isCollapsed && (
                <>
                  <div className="grid flex-1 text-left text-sm leading-tight">
                    <span className="truncate font-semibold">{activeTeam?.name}</span>
                    <span className="truncate text-xs">{activeTeam?.plan}</span>
                  </div>
                  <ChevronsUpDown className="ml-auto" />
                </>
              )}
            </SidebarMenuButton>
          </DropdownMenuTrigger>
          <DropdownMenuContent
            className="w-[--radix-dropdown-menu-trigger-width] min-w-56"
            align="start"
          >
            <DropdownMenuLabel>Teams</DropdownMenuLabel>
            {teams.map((team, index) => (
              <DropdownMenuItem
                key={team.id}
                onClick={() => onSwitchTeam(team.id)}
                className="gap-2"
              >
                {/* Logo */}
                <div className="flex size-6 items-center justify-center rounded-sm border">
                  {team.logo ? (
                    <img src={team.logo} alt={team.name} className="size-4" />
                  ) : (
                    <span className="text-xs">{team.name[0]}</span>
                  )}
                </div>
                {team.name}
                <DropdownMenuShortcut>⌘{index + 1}</DropdownMenuShortcut>
              </DropdownMenuItem>
            ))}
            <DropdownMenuSeparator />
            <DropdownMenuItem onClick={onCreateTeam}>
              <Plus className="size-4 mr-2" />
              Add team
            </DropdownMenuItem>
          </DropdownMenuContent>
        </DropdownMenu>
      </SidebarMenuItem>
    </SidebarMenu>
  );
}
```

#### 2. `Layout.tsx` - 数据与逻辑中心

```typescript
import { TeamSwitcher } from './team-switcher';
import { trpc } from '../lib/trpc';
import { useQueryClient } from '@tanstack/react-query';
import { useNavigate } from 'react-router-dom';
import { toast } from 'sonner';

export function Layout() {
  const queryClient = useQueryClient();
  const navigate = useNavigate();

  // 获取组织列表
  const { data, isLoading } = trpc.organization.listMine.useQuery();
  const organizations = data?.organizations ?? [];

  // 获取当前活跃组织（从 Better Auth session）
  const { data: session } = useSession();
  const activeOrgId = session?.user?.activeOrganizationId;

  // 切换组织 Mutation
  const switchMutation = trpc.organization.setActive.useMutation({
    onSuccess: async () => {
      // 严格清理：清空所有缓存（防止跨租户数据泄露）
      await queryClient.resetQueries();

      // 导航到首页
      navigate('/', { replace: true });

      // 成功提示
      toast.success('Organization switched successfully');
    },
    onError: (error) => {
      toast.error(error.message || 'Failed to switch organization');
    }
  });

  const handleSwitchTeam = async (teamId: string) => {
    if (teamId === activeOrgId) return; // 避免重复切换
    await switchMutation.mutateAsync({ organizationId: teamId });
  };

  const handleCreateTeam = () => {
    navigate('/organizations/new');
  };

  // 转换数据格式
  const teams = organizations.map(org => ({
    id: org.id,
    name: org.name,
    logo: org.logo ?? '',
    plan: mapRoleToPlan(org.role), // 'OWNER' → 'Enterprise'
  }));

  return (
    <SidebarProvider>
      <Sidebar collapsible="icon">
        <SidebarHeader>
          <TeamSwitcher
            teams={teams}
            activeTeamId={activeOrgId}
            isLoading={isLoading}
            onSwitchTeam={handleSwitchTeam}
            onCreateTeam={handleCreateTeam}
          />
        </SidebarHeader>
        <SidebarContent>
          {/* 现有菜单渲染逻辑 */}
        </SidebarContent>
        <SidebarFooter>
          <NavUser user={userData} onLogout={logout} />
        </SidebarFooter>
      </Sidebar>
      {/* ... */}
    </SidebarProvider>
  );
}

// 辅助函数：映射角色到套餐类型
function mapRoleToPlan(role: string): string {
  switch (role) {
    case 'OWNER': return 'Enterprise';
    case 'ADMIN': return 'Pro';
    case 'MEMBER': return 'Free';
    default: return 'Free';
  }
```

### 状态管理策略

#### 当前组织状态来源

**方案**：使用 Better Auth 的 `useSession` hook

```typescript
const { data: session } = useSession();
const activeOrgId = session?.user?.activeOrganizationId;
```

**理由**：
- ✅ 与后端 session 保持一致
- ✅ 切换后自动同步（Better Auth 自动刷新 session）
- ✅ 无需额外状态管理

#### 缓存清理策略

**执行时机**：`setActive` mutation 成功后

```typescript
// 严格清理（推荐）
await queryClient.resetQueries(); // 清空所有查询缓存

// 或选择性清理（如需保留某些全局数据）
queryClient.invalidateQueries({ queryKey: ['menus'] });
queryClient.invalidateQueries({ queryKey: ['permissions'] });
queryClient.invalidateQueries({ queryKey: ['settings'] });
```

**理由**：防止跨租户数据混淆（如租户 A 的菜单显示在租户 B）
